About This Website

This website is a fictional character gallery based on the Harry Potter universe. I chose this theme because the characters are well known, easy to recognize, and each character has interesting details such as a house, role, personality, and story.

The purpose of this website is to practice building a static web application using HTML, CSS, and Bootstrap. The home page shows a gallery of characters, the details page gives more information about one character, and the form page allows users to enter information for a new character.

I used Bootstrap to create a clean layout with navigation, cards, buttons, and responsive design. I also added custom CSS to change the colors and make the website fit the Harry Potter theme.
