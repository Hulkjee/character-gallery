About This Website

I created this websire as a part of my CSC340 assignment. It ia a harry Potter character Gallery where users can learn about some of the most popular characters from the Harry Potter series. I chose Harry Potter becasue I enjoy the movies, and i think the characters are interesting. Each character has different personality, house, and role, which makes them a good choice for this project.

This website was built using HTML, CSS, and Bootstrap. Bootstrap helped me create the navigation bar, character cards, buttons, and page layout. I also used my own CSS to change the colors and make the website match the Harry Potter theme.

The website includes a home page with five characters, a details page with more information about Harry Potter, an about page, and a form where users can add a new character. The search bar and form are only for the layout because this is a static website and does not have a backend yet.
